# Agent Aide (v0.5.0 - MVP)

An enterprise-grade, offline-first mobile application designed specifically for real estate agents to manage properties, track guest stays, and automate client communication locally.

## Current MVP Features
* **Relational Local Database:** Powered by Expo SQLite in synchronous WAL mode for zero-latency, offline-first data persistence.
* **Regex Parsing Engine:** Instantly extracts flight numbers, dates, and guest counts directly from device clipboard text.
* **WhatsApp Deep Linking:** Automatically generates formatted, professional itineraries and launches the native WhatsApp client to share with guests.
* **Modern UI/UX:** Built with React Native, Expo Router (Bottom Tabs), and strict TypeScript typing.

## Architecture & Tech Stack
* **Framework:** React Native / Expo Router v6
* **Language:** TypeScript
* **Local Storage Engine:** Expo SQLite
* **Target Platforms:** iOS & Android (Offline First)
